## Как собрать для линукса
Для сборки понадобятся пакеты libqt4-dev и qt4-qmake. Версия Qt 4.8.x.

    git clone git://github.com/theanonym/onechan-plusoner.git
    cd onechan-plusoner
    qmake
    make
    make clean
    ./plusoner
